# Hagiographies Local Map

Interactive vector map for BantuLex linguistic data, with an integrated BLR sound search (Klankzoeker).

## Features
- **MapLibre GL JS**: Smooth vector map rendering.
- **PMTiles**: Local basemap tile archive.
- **Hagiographies tab**: Click map markers to see hagiographies per variety.

## Project Structure
- `index.html`: Original BantuLex map interface.
- `bantulex_klankzoeker.html`: Extended map with integrated BLR Klankzoeker.
- `data/`: GeoJSON and PMTiles data files.
  - `bantulex_entries.geojson`: Exported linguistic data (generated by export scripts).
  - `blr_catalog.json`: BLR phonological index for the Klankzoeker (generated by `export_blr.py`).
  - `africa.pmtiles`: Basemap tile archive.
- `scripts/`:
  - `export_bantulex.py`: Export linguistic data from SQLite to GeoJSON (original, for `index.html`).
  - `export_blr.py`: Export BantuLex + BLR phonological data to GeoJSON and `blr_catalog.json` (for `bantulex_klankzoeker.html`).
  - `download-world-pmtiles.js`: Script to download/extract PMTiles basemaps.
- `Dockerfile`: Caddy-based container for serving the map.

## Maintenance

### Updating BantuLex data (original map)
```sh
uv run python scripts/export_bantulex.py
```

### Updating BantuLex + BLR sound data (Klankzoeker)
```sh
uv run python scripts/export_blr.py
```
This overwrites `data/bantulex_entries.geojson` (with BLR fields added) and generates `data/blr_catalog.json`.

### Changing Map Area
To re-extract the basemap (e.g., if you need a larger area):
```sh
node scripts/download-world-pmtiles.js --bbox=S,W,N,E --output=africa.pmtiles --maxzoom=8
```
(Requires `pmtiles` CLI installed on host or via Go).
